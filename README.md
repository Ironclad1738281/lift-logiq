# LiftLogIQ

LiftLogIQ is a full-stack fitness analytics platform that helps users track workouts, recovery, soreness, sleep, pain, and training load. The app analyzes training patterns to generate recovery insights and injury-risk scores, helping lifters make smarter decisions about when to push, rest, or adjust their training.

## Core Features

- Log workouts with exercises, sets, reps, weight, muscle group, and intensity
- Track recovery metrics like soreness, sleep, pain, and energy
- View weekly training volume and body-part distribution
- Monitor soreness, sleep, and pain trends over time
- Generate rule-based recovery and injury-risk scores
- Display progress and risk insights in a dashboard
